#OC_P5
## A blog with a back office

### Features :

* For visitor :
  * See list of posts
  * See post
  * See list of comments
  * Drop comment
  * Pagination
  
* For admin :
  * All the above +
  * Write new post
  * Update post
  * Delete post
  * Checks comments

### Libraries :

* symfony/var-dumper
* filp/whoops
* swiftmailer/swiftmailer
